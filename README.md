## Установка

Загрузите этот репозиторий и выполните

```bash
npm i
```

## Использование

### Локальный сервер разработки

```bash
npm run serve
```

После запуска страница будет доступна на `localhost:8080`

### Компиляция продакшен сборки

```bash
npm run build
```

После компиляции сборка будет в директории `dist`
